== Description ==
Don't get lost again while making deliveries get the exact location of your customer  you need to deliver to!

With Autolocation plugin you can geolocate your customers on woocommerce  checkout page, by adding their coordinates on the checkout form.

The coordinates will be visible in the order and also in the order emails.

You will be sure that your delivery service will know exactly where to deliver the goods


== Installation ==

This section describes how to install the plugin and get it working.


1. Upload autolocation-checkout folder to the ./wp-content/plugins/. directory
2. Activate the plugin through the 'Plugins' menu in WordPress
